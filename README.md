# 📚 Biblioteca Virtual

Sistema de gerenciamento de biblioteca desenvolvido em **C# com .NET**, com funcionalidades completas de cadastro, consulta e controle de acervo bibliográfico. Projeto acadêmico desenvolvido como parte do **Projeto Integrado Multidisciplinar (PIM)** do curso de Análise e Desenvolvimento de Sistemas da UNIP — 2024/2.

---

## 🖥️ Sobre o Projeto

O sistema foi desenvolvido com o objetivo de simular o gerenciamento de uma biblioteca, permitindo o controle de livros, usuários e empréstimos por meio de uma interface simples e funcional.

O projeto aplica conceitos fundamentais de **Programação Orientada a Objetos**, organização de soluções Visual Studio e boas práticas de desenvolvimento de software com C# e .NET.

---

## ✅ Funcionalidades

- 📖 Cadastro, listagem, edição e remoção de livros
- 👤 Cadastro e gerenciamento de usuários
- 🔄 Controle de empréstimos e devoluções
- 🔍 Consulta ao acervo por título, autor ou categoria
- 💾 Persistência de dados

## 🚀 Como Executar

### Pré-requisitos

- [.NET SDK](https://dotnet.microsoft.com/download) instalado
- [Visual Studio](https://visualstudio.microsoft.com/) ou [VS Code](https://code.visualstudio.com/)

### Passos

1. **Clone o repositório**
```bash
git clone https://github.com/GuiMRDS/Bibloteca_Vitual.git
```

2. **Acesse a pasta do projeto**
```bash
cd Bibloteca_Vitual
```

3. **Abra no Visual Studio**

Abra o arquivo `Bibloteca_Vitual.sln` diretamente no Visual Studio e execute com `F5` ou `Ctrl+F5`.

**Ou execute via terminal:**
```bash
dotnet run --project Bibloteca_Vitual
```
